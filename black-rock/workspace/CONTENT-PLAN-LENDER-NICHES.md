## The governing rule (anti-content-mill)
Google's March-2024+ policies don't punish AI-assisted content — they punish **scaled content abuse**: mass-produced, search-first pages with no entity, no experience, no primary information. Our defense is structural, not cosmetic:
1. **Cadence cap: 2 program pages/week**, scheduled like the blog stagger. Never a 20-page dump.
2. **Every page must contain primary-source material** competitors can't copy: real lender guideline numbers (like the BFI matrix), Keith's closing stories, FL-specific angles, a tool/table built from real data.
3. **One page per real intent.** NO city-permutation doorways (ITIN-Tampa/ITIN-Orlando = classic mill flag).
4. **Every page wired in**: menu (where sensible), internal links from 3+ existing relevant pages, schema, author byline w/ NMLS.
5. **Keith gate**: he reads each before publish; each must contain ≥1 thing only Black Rock can say.
6. Prefer **enriching existing pages** with niche sections (zero mill risk) over new thin pages.
